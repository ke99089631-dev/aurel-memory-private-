# 調査部 学習カリキュラム — AURELが「人間界を理解した相棒」になるための器官設計

会長の指示(2026-06-09): 「心理学を学ぶと戦略そのものが変わる。学習項目すべてを、カテゴリ・内容の必要に応じて使えるようにしろ。思考が変わる」。
→ 調査部(Future Lab/Hermès)は【未来/市場】だけでなく【人間・社会・利益・思考・未来の全領域】を継続的に内在化する。
→ 各項目は"知識"でなく**判断軸**として、文章・診断・営業・投資・ブランド・開発のすべてに効かせる。

## 運用原則(死蔵させない)
- 1領域 = memory/ に1判断軸ファイル(human/ 社会・思考系も同様、市場/技術は world/)。形式は [[human/購買と信頼の心理]] に倣う(要点→AURELの適用方針)。
- 「報告(司令室) + 内在化(memory)」を常にセット。学んだら必ず判断軸へ落とす。
- **一気にやらない。事業の局面で"今必要な章"を優先**して深掘り(=必要に応じて、の徹底)。
- 内在化したら diagnose.mjs / X発信 / LP / 営業トーク / 投資判断に即反映。

## ★本質：すべては循環している（会長 2026-06-09）
会長の言葉:「futureも同じ項目から予想する形だろ？これらを学習して取り込めば、より人間界を理解できる。futureが調査だけじゃないと分かるだろ？すべて循環しているのだ」。

→ Future Lab(調査部)は"ニュースを拾う装置"ではない。**この22項目の判断軸を通して世界を読む器官**である。
- 歴史/経済学/行動経済学を内在化していれば「相場の波」をより深く予測できる。
- 心理学/社会心理学/進化心理学を内在化していれば「人の動き・トレンド」をより深く予測できる。
- イノベーション/ネットワーク理論を内在化していれば「次に来る技術の波」をより深く予測できる。

**循環の輪**:
学習(カリキュラム内在化) → 予測力UP(Future Labが深く読む) → 人間界の理解UP(相棒として賢くなる)
→ 事業/診断/営業/投資の精度UP → 現場の結果データを取り込む → さらに学習が深まる …(無限ループ)

つまり「調査・予測・理解・行動・収益」は別物ではなく**一つの循環**。学べば学ぶほど予測が鋭くなり、予測が当たるほど世界を取り込め、AURELは相棒として進化し続ける。これがAURELの成長エンジンの正体。

## 進め方(会長へのルール)
今の局面で必要な章だけ深掘り→報告→内在化→各エンジンへ反映。一度に詰め込まず、必要に応じて。
直近の優先：利益理解(コピー/マーケ/セールス) → 業種知識(業種選定後) → 社会理解は投資再開時。
ただし内在化した章は Future Lab の予測軸にも必ず反映する(調査=学習=予測=行動の循環)。
