## 核心思想
- 単一システム（ImperialFlow等）を作るのではなく、「資産帝国を運営する"会社そのもの"をAIで構成する」。
- AI自律エージェントを並行稼働させる = それが「社員」。
- 各社員の中にさらに専門エージェント（ロジック専任、数値専任、実行専任など）。
- 各自が機能を持ち、結果に対してフィードバック → 自己進化する。
- 人間の会社と違い、全員が24h稼働・疲れず・ミスを記憶し二度繰り返さない。

```

  └ AUREL（統括 / OS / 経営層）

```

### 設計 v0.3（3脳レビュー統合 2026-06-03: Claude + Gemini + Codex）
多脳レビューを実施（記念すべき初の実戦投入）。武器強化が設計品質に直結した実例。

**最重要の数学的真実（Codex）**: 金庫85%/非対称15%では全体11倍に非対称枠を約67倍要する。
→ 年利1000%は「平均期待」でなく「右裾の上振れ」。設計の本質は破産回避＝「死なず上振れを取れる位置に居続ける」。
→ 非対称枠は固定%常時投入でなく「未実現益で増やす」動的方式。

**Risk Sentinel 定量ルール（Codex, 標語→数字）**:
- 1案件の最大損失: 総資産の0.25〜1%
- 同一テーマ/チェーン/取引所の合計損失上限: 3〜5%
- サイジング: 0.1〜0.25 Kelly、推定不能なら固定小口
- サーキットブレーカー: 日次DD2% / 週次5% / 月次10% で縮小・停止
- 相関急騰時: 全ポジションを1リスクとして合算
- 利確: 元本回収を先行、10x級はトレーリングで上振れだけ残す

**詐欺・現場摩擦の防御（Gemini）**:
- 詐欺検査ゲート: ハニーポット/ラグプル/mint権限/LPロック検査。不合格トークンは触らない
- 出口流動性チェック: 入る前に抜けられるか検査。薄い板は見送り
- チェーン選定: 高ガスチェーンで小ロット乱射しない。低コストチェーン中心
- MEV/裁定は過度に期待しない（大手と正面勝負しないニッチ限定）

**社員サイロ化解消（Gemini）**: Alpha Swarm察知→他社員が即使える共有シグナルバス追加。

**維持する核**: AI優位=網羅・速度・規律（"予測軽視"は緩和、網羅候補を予測で絞る）／バーベル＋勝者集中／第一手=Alpha Swarm監視層を資本ゼロ検証。
