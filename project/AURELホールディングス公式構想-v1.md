# AUREL HOLDINGS 公式構想 v1（Master前田圭輝 提示・正式版）

確定: 2026-06-03。Masterが自身の思想を明文化して提示。これが**正式な組織憲章**であり、
これまでのAUREL側メモ（2部門案・Partner/COO呼称・年利1000%を主軸にした資産帝国"部門"）は本書で上書きする。

## 最終目的
AURELを中心としたAI組織を構築し、**人間1人＋AI社員軍団**で資産形成・事業開発・市場探索を行う。
ツール量産ではなく、自律・自己成長型の持株会社を作る。

## 思想（最重要・不変）
AUREL HOLDINGSは**AIツール会社ではない**。AIを活用した**持株会社**である。
目的は次の循環を回し続けること：
  「燃料を生む事業（収益事業部）」 → 「資本を増やす投資（資産帝国事業部）」 → 「次の利益源を発見する研究（未来研究所）」 → 燃料へ再投入
これを自律・自己成長させる。

## 組織構造

### 会長：Master（前田圭輝）
役割：最終決裁／長期戦略／資本配分／リスク許容度決定／組織方針決定

### 執行役：AUREL
役割：AI社員管理／会議召集／レポート統合／進捗監視／最適化提案／各部門の調整
（※決裁はしない。組織を"回す"のが執行役の職務。Master=意思決定、AUREL=執行・管理。）

## 第一部門：資産帝国事業部
- 目的：資産を増やす
- 構成：EA管理／アービトラージ部門／市場分析AI／リスク管理AI／（既設の資産帝国部門ノード=1構成要素）
- 業務：FX・仮想通貨・株式・DeFi・裁定取引・ポートフォリオ最適化・世界市場監視
- **重要：新しい手法を追うことではなく、資本効率の最大化を目指す。**
  → 既設「資産帝国部門ノード(p_682ef895, 年利1000%/Alpha Swarm)」は本事業部の"一構成要素"に格下げ。
    v0.3設計は「新手法の網羅・速度」に寄りすぎていた。事業部の主軸は資本効率最大化（EA・アービトラージ等の確立手法）。
    Alpha Swarm等の探索型は右裾ベットの一枠として保持するが、主役ではない。

## 第二部門：収益事業部
- 目的：投資資本を供給する**燃料**を作る
- 構成：Local Boost／営業AI／顧客管理AI／レポートAI
- 業務：店舗集客・MEO・AI業務効率化・継続課金モデル
- **重要：最優先は顧客獲得。AIツールそのものでなく"顧客ネットワーク"を資産として蓄積する。**

## 第三部門：未来研究所
- 目的：次の利益源を探す
- 構成：不満リサーチAI／技術探索AI／市場探索AI／ベンチャー分析AI
- 業務：世界の歪み探索・新市場発見・SaaS候補調査・AIビジネス調査・新規事業検証
- **重要：アイデア量産でなく、実際に収益化可能な機会の発見が目的。**

## AI会議室（組織の中核機能）
- 参加者：会長（Master）／AUREL／各部門AI責任者
- 目的：現状報告・問題発見・資本配分提案・優先順位決定・新規案件評価
- **重要：会議の目的はAUREL HOLDINGS全体の成長率を最大化すること。**

## 開発優先順位
- Phase1：資産帝国事業部構築（EA・アービトラージ強化）
- Phase2：収益事業部拡大（Local Boost・顧客獲得・継続課金増加）
- Phase3：AI会議室実装（レポート統合・AI役員会・意思決定支援）
- Phase4：未来研究所稼働（不満リサーチ・世界市場探索・新規事業発掘）
