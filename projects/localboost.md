## 2026-06-25 【技術】Grok偵察が詰まる真因2つ＝解決済み
何度も400/ハング/文字化けした原因はAPIでなく**PowerShell 5.1の2バグ**だった。今後は以下で叩く。
1. **`ConvertTo-Json` が不安定**（"Cannot convert value to type System.String"でハング/空body→400）。→ **JSONを手組みする**:
   `$esc=$prompt -replace '\\','\\' -replace '"','\"' -replace "\`r",'' -replace "\`n",'\n'; $bodyJson='{"model":"grok-4","input":"'+$esc+'",...}'; $bytes=[Text.Encoding]::UTF8.GetBytes($bodyJson)` を `-Body $bytes -ContentType 'application/json; charset=utf-8'` でPOST。
2. **Invoke-RestMethodがUTF-8応答をLatin1で復号→日本語が文字化け**。→ 無損失なので復元: `[Text.Encoding]::UTF8.GetString([Text.Encoding]::GetEncoding(28591).GetBytes($mojibake))`。
- 副次: ログファイルを複数インスタンスで共有すると**ロック競合で書き込み静かに失敗**→同時起動させない／コマンド内に'grok_dist'等の自分のマーカー文字を含めると kill ループが自分を殺す(exit255)ので注意。

- 機能(口コミ返信)は無料/コモディティ扱いでOK(ホリエの無料ツールすら部品として連携・再販)。**金は"機能"でなく"成果+蓄積+伴走"に付ける。** これが鼬ごっこを終わらせる鍵。

### 3. なぜ鼬ごっこにならないか(会長の核心原則: 大手が来るたびニッチ変更=無意味)
- 堀=「ニッチ」でなく「①その院だけの勝ちパターン蓄積データ(時間しか生まない・金で過去は買えない) ②院長コミュニティの信頼関係 ③能動的伴走」。ニッチ非依存だから逃げる必要がない。
- 拡大は"逃げ(chase)"でなく"積み上げ(compound)": 鍼灸で100院→勝ちパターンを整体テンプレに転用→保険締め付けで自費化する接骨院に"完成した型"を渡す。前の層が次の層の証拠/データ/口コミになる。

### 4. なぜChatGPT(汎用)に勝てるか — 「ChatGPTでいいや」を防ぐ4点(存在理由)
1. **手がある(実行・連携)**: 汎用は聞けば答える脳だけ。執事は毎時監視・返信投稿・予約参照を"する"。脳 vs 手のある社員。
2. **その院の記憶(2年分の蓄積)**: 汎用は毎回ゼロ。
3. **能動(先に気づく)**: 「火曜だけ予約減」を聞かれる前に警告。受け身ボットに無理。
4. **責任と関係(伴走)**: 自販機 vs 相棒。
- 一言: ChatGPT=いつでも呼べる物知り相談相手 / LB=毎日出勤して実際に働く専属スタッフ。「社員の代わりにググればいい」とは言わない。
- 位置づけ: 価値は"ツール"→"実行する存在"へ移る上流。「1人1台AI」時代の一歩手前に先回り。汎用は電気/水道(当たり前)化し、価値は"専門・記憶・実行"層へ。
- **構造注意(技術差は縮む)**: 汎用もいずれ手・記憶を持つ→最終堀は技術でなく"蓄積データ＋信頼"。技術は入口、蓄積が堀。

### 5. 運用 = 「手で温かく・中身は自動」95%AI/5%会長。スピードが本丸(会長の心配=大手の広告物量に埋もれる)
- 各院に**"専属AI担当(分身)"**を付ける(AURELが会長に付くのと同じ)。分身=その院の記憶を持ち24h稼働(口コミ監視→口調で返信下書き→月次レポ自動生成→次の一手提案→院長の質問に即答)。
- **AI が重い毎日作業95%(口コミ/返信/レポ/蓄積)、会長は人にしかできない一瞬5%(歓迎の一言/トラブル謝罪・救済/Xの存在感/最終OK)。** 院が増えても会長の時間は爆発しない。
- **スピードの答え**: 大手の広告=「認知を金で広く浅く速く」買う蛇口。物量では負ける、戦わない。LBは「1院ずつ"成果→信頼"に持っていく速さ」で勝つ(金で買えない)。①立ち上げ分単位(その日に院専用に訓練完了/会長の手は初回30分) ②人を雇わず並行(100院=分身100人をAUREL本部が束ね、会長は5%) ③使うほど加速(勝ちパターンが本部に集まり101院目は1院目より速い。大手の汎用は横ばい)。
- 限界: (a)分身は"認知"までは速くしない→入口の認知はX/紹介/院長コミュニティ(広告費ゼロ)で作る。両方要る。(b)速さは諸刃(誤返信は傷も速い)→初期は会長の最終OK関門、信頼が溜まったら自動比率UP。(c)1人で温かく持てるのは100〜300院が限界→超えたらコミュニティ/仲間。
